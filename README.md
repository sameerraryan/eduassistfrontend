#  EduAssist — AI-Powered Teaching Platform

Welcome to the frontend repository of EduAssist .This is a modern, responsive, and highly interactive landing page designed for a conceptual AI-powered educational toolkit. 

The core idea behind this project is to envision a platform where teachers can teach *smarter, not harder*. By automating time-consuming tasks like lesson planning, quiz generation, and content simplification, educators can get back to what they do best: connecting with their students.

## What is this project?
EduAssist is a purely front-end web project built from scratch. The goal was to design a sleek, premium SaaS-style landing page that feels alive and interactive, all without relying on heavy JavaScript frameworks (like React or Vue) or external animation libraries. 

##The Cool Stuff (Key Features)

* **Interactive AI Chat Demo**: Easily my favorite part of the site! Written entirely in vanilla JavaScript, the "AI Tools" section simulates a real-time AI assistant typing out lesson plans, quizzes, and student feedback. 
* **Modern UI/UX Design**: Built with a dark/light mixed aesthetic, featuring glassmorphism (frosted glass effects), floating parallax cards on the hero section, ambient background glows, and a carefully chosen color palette (Navy, Amber, and Teal).
* **Scroll Animations & Micro-interactions**: Uses the native `IntersectionObserver` API for buttery-smooth fade-in animations as you scroll down the page. It also includes dynamic stat counters that tick up to their target numbers, and neat 3D tilt effects on the feature cards.
* **Fully Responsive**: Adapts seamlessly to desktop, tablet, and mobile screens. Includes a custom-built hamburger navigation menu for smaller devices.
* **Dynamic Pricing Toggle**: A sleek monthly/yearly switch that smoothly animates the price changes when clicked.

## Under the Hood (Tech Stack)

* **HTML5**: Clean, semantic structure.
* **CSS3**: Extensive use of CSS Custom Properties (variables) for easy theming, CSS Grid & Flexbox for layout, and custom `@keyframes` for continuous ambient animations. Zero external CSS libraries were used.
* **Vanilla JavaScript (ES6+)**: Handles all the logic—from the typing simulations and scroll observers to the pricing math and mobile navigation.
